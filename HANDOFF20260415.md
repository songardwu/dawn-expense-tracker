## 架構筆記

- **無 ORM** — 全部用 psycopg2 raw SQL，SQL 集中在 app.py 各路由內
- **Session 機制** — Flask signed cookie，不依賴 server-side store，適合 serverless
- **密碼加密** — werkzeug.security（PBKDF2），非明碼
- **資料隔離** — SQL WHERE 層級，非中間件過濾
- **Excel** — pandas + openpyxl，BytesIO 串流輸出，不落地檔案
- **RWD** — CSS media query 769px 分界，手機卡片 / 桌面表格雙版面

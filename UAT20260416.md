## 四、UAT 驗收流程

### Step 1：桌面版巡覽（管理員）

請在 Chrome/Edge 開啟以下 5 頁，逐頁確認：

1. **登入頁** — https://dawn-expense-tracker.vercel.app/login
   - [ ] 暖奶油背景、Serif 標題、微圓角按鈕
   - [ ] 輸入框可正常聚焦

2. **清單頁** — https://dawn-expense-tracker.vercel.app/
   - [ ] Bento Grid 佈局，表格清晰
   - [ ] 篩選功能正常
   - [ ] 加總面板顯示正確

3. **新增提報** — https://dawn-expense-tracker.vercel.app/new
   - [ ] 表單卡片式佈局
   - [ ] 所有欄位可正常填寫
   - [ ] 送出功能正常

4. **廠商管理** — https://dawn-expense-tracker.vercel.app/vendors
   - [ ] 廠商列表正常顯示
   - [ ] 新增/編輯/刪除功能正常

5. **帳號管理** — https://dawn-expense-tracker.vercel.app/users
   - [ ] 帳號列表正常
   - [ ] 新增帳號功能正常

### Step 2：手機版巡覽（設計師）

請用手機 Safari/Chrome 開啟，重點確認：

- [ ] `/new` 新增提報 — 輸入框點擊無自動縮放
- [ ] `/` 清單頁 — 卡片式列表正確堆疊
- [ ] 字型正確載入（標題 Serif、內文 Sans）
- [ ] 底部 safe area 無遮擋

### Step 3：效能確認

- [ ] Chrome DevTools → Network → Throttle: Fast 3G → 載入 `/new`
- [ ] FCP ≤ 2s
- [ ] 頁面完整渲染 ≤ 3s

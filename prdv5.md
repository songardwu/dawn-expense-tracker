# PRD V5 — UI 設計改版（暖大地色系 + Bento Grid）

## 背景

目前系統使用 Apple 極簡風格（冷白背景、SF Pro 字型、毛玻璃 navbar），功能完善但視覺上缺乏室內設計公司的品牌識別。V5 將 UI 改版為暖大地色系，搭配人文感字型與 Bento Grid 排版，營造健康、無壓力且具備高級感的居家氛圍，同時維持完美的手機自適應和無障礙標準。

## 設計目標

- 利用「暖大地色系」營造健康、無壓的居家氛圍
- 強化「設計師」職人形象（人文感字型 + 自然材質色）
- 全頁面 Bento Grid 排版，簡潔、層次分明、具備信任感
- 符合 WCAG AA 無障礙標準
- 保留 Dark Mode（用大地色系重新設計暗色版本）
- 維持手機端（51% 流量）完美操作體驗

## 2. 字體規範 (Typography)

### 字型選擇

| 角色 | 字型 | 字重 | 用途 |
|------|------|------|------|
| 大標題 (Heading) | Noto Serif TC（思源宋體） | Bold 700 | 首頁 Slogan、頁面大標、品牌名稱 |
| 內文 (Body) | Noto Sans TC（思源黑體） | Regular 400 | 作品介紹、服務流程、表單文字 |

### 排版參數

- **行高 (Line Height)**：1.6 ~ 1.8（增加空氣感，減少閱讀壓力）
- **手機端標題字級**：不低於 20px（宋體在小螢幕 < 16px 時筆畫偏細）
- **手機端內文字級**：16px（Noto Sans TC，確保易讀性）

### 字型載入策略

- `font-display: swap`（避免白屏等字型）
- 只載入需要的字重（Serif 700 + Sans 400）
- Google Fonts `&subset=chinese-traditional` 限制範圍
- 設定 fallback：`"Noto Serif TC", "PMingLiU", serif` / `"Noto Sans TC", "Microsoft JhengHei", sans-serif`

## 3. 視覺元素與組件 (UI Components)

### 佈局結構：Bento Grid（便當盒排版）

- **套用範圍**：全部頁面（清單頁、新增頁、廠商頁、帳號管理頁）
- **桌面端**：多欄網格排列，資訊密度高
- **手機端**：自動堆疊為單欄卡片，維持觸控友善
- **表單頁例外**：表單本身為線性流程，不強制格子排版，但外框可用 Bento 卡片風格包裹

### 按鈕設計 (Buttons)

| 屬性 | 規格 |
|------|------|
| 圓角 | `border-radius: 4px ~ 6px`（微圓角） |
| 主按鈕 | 深石墨底 `#333` + 白字 |
| 主 CTA | 森林深綠底 `#4A5D4E` + 白字 |
| 次要按鈕 | 胡桃棕底 `#7A6652` + 白字 |
| 危險按鈕 | 維持紅色系（刪除等破壞性操作） |
| 過渡動畫 | `transition: all 0.3s ease`（柔和淡入淡出） |
| 觸控區域 | 最小高度 44px（iOS Human Interface Guidelines） |

### 卡片設計

- 背景色：`#FFFFFF`（純白，與暖奶油底形成層次）
- 邊框：`1px solid` 淡灰或無邊框搭配微陰影
- 圓角：`8px ~ 12px`
- 內距：`16px ~ 24px`

### Navbar

- 保留固定頂部導航列
- 風格由毛玻璃改為大地色系實底（暖奶油或深石墨）
- 品牌名 "Dawn" 使用 Noto Serif TC

## 5. 響應式設計 (Responsive)

### 斷點維持不變

| 斷點 | 裝置 | 行為 |
|------|------|------|
| < 390px | 小螢幕手機 | 單欄、更緊湊間距 |
| < 768px | 手機 | Bento Grid 堆疊為單欄卡片 |
| 768px+ | 平板/桌面 | Bento Grid 多欄 |
| 1200px+ | 大螢幕 | 更寬鬆間距、更多欄 |

### 手機優化重點

- 觸控區域最小 44px
- 表單 input 不小於 16px（防止 iOS 自動縮放）
- iPhone safe area 繼續支援 `env(safe-area-inset-*)`
- Noto Serif TC 標題不低於 20px

---

## 6. Dark Mode（大地色系暗色版）

- 觸發方式：`prefers-color-scheme: dark` media query（跟隨系統）
- 背景：深色大地色調（如深棕 `#2A2520` 或深墨 `#1E1E1C`）
- 文字：暖白 `#E8E4DF`
- 強調色：森林綠和胡桃棕維持不變或微調亮度
- 卡片：深色表面色（如 `#3A3530`）
- 保持所有對比度符合 WCAG AA

---

## 7. 效能考量

- **字型載入**：`font-display: swap` + 僅載入需要的字重（避免 CJK 全量載入）
- **CSS 改動範圍**：僅修改 CSS 變數和字型，不新增 JS，不改資料流
- **表單頁速度**：新增出帳頁是最簡單的頁面，Bento Grid 不影響表單性能
- **不影響 69 項功能測試**：純 CSS/模板改動，功能邏輯零修改
